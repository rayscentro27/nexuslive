# Nexus — Mission (read-only context)

**Mission:** Help people at scale across credit, funding, opportunity, and
trading-education — and build a real business doing it — while operating safely
and provably.

**Operating principle:** Build aggressively; gate only truly risky/outward-facing
actions. Show proof (real assets, real outcomes), not just dashboards.

**Current shape (V1):**
- **Proof Automation Engine** — turns an idea/scenario into track-specific assets
  (landing page, checklists, lead magnet, DM/email drafts, 30-day plan, proof report).
- **7 scouts** — credit, funding, opportunity, trading, marketing, metrics,
  ai_improvement. Template-driven today → findings flagged "needs live verification."
- **Showroom** — every output is a reviewable asset (needs_review → approved/revise).
- **Command bot (TheChoseone)** — status, approvals, batch approvals, controls.
- **Hermes Mobile (new, this build)** — read-only conversational layer.

**Money model (realistic):** one focused paid offer first — credit-readiness
review/consult — to a warm audience; prove a result; upsell funding readiness.
No automated payments yet (gated).

**Safety posture (always on):** email allowlist-only, IG queue-only, Oanda demo,
no public posts, no payment automation, approved_live OFF, no secrets exposed,
no auto-approval without an explicit Ray batch command.
