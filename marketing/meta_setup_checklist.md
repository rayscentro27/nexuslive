## Safety Rules

| Rule | Reason |
|------|--------|
| No auto-posting tools connected | Risk of platform violation, account suspension |
| No third-party login apps | Security risk — credentials exposure |
| 2FA on all accounts | Prevent unauthorized access |
| No shared passwords | Each account owned by one admin |
| No purchased followers | Violates platform terms, damages engagement rate |
| No DM automation | Violates platform terms; potential ban |
| Review API permissions before granting | Prevent unintended data access |
