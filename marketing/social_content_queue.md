## Status Definitions

| Status | Meaning |
|--------|---------|
| `idea` | Concept captured, not yet scripted |
| `drafted` | Script or outline written |
| `reviewed` | Reviewed by owner, feedback given |
| `approved` | Ready to record/design |
| `scheduled manually` | Asset ready, posting date set manually |
| `published manually` | Posted by hand |
| `repurposed` | Adapted from existing content for another platform |

---

## Queue Fields

| Field | Description |
|-------|-------------|
| ID | Unique content ID (e.g. C001) |
| Idea/Title | Working title or concept |
| Platform | YouTube / Reels / TikTok / LinkedIn / X / Facebook |
| Format | Short (< 60s) / Long (> 3min) / Post / Carousel / Thread |
| Hook | Opening line or visual hook |
| Script | Link to draft or inline summary |
| CTA | What action the viewer should take |
| Status | See Status Definitions above |
| Owner | Who is responsible for this piece |
| Publish Date | Target date (manual scheduling) |
| Asset Needed | Video recording / Graphic / Voiceover / B-roll |
| Disclaimer | Yes/No — compliance disclaimer required |
| Notes | Any special considerations |

## Content Templates

### Short-Form Script Template (Reels / TikTok / Shorts)

```
HOOK (0–3s): [Surprising statement or question]
PROBLEM (3–10s): [Who this is for / what pain it solves]
SOLUTION (10–45s): [3 key points, numbered]
CTA (45–60s): [One clear action — comment, follow, link in bio]
DISCLAIMER (if needed): [Results vary. Educational only.]
```
