# Revenue Loop Finalization

## Current Status
Partial: components exist (content, lead magnets, newsletters, referrals, subscriptions) but loop orchestration is fragmented.

## Required Connected Loop
Content -> lead magnet -> onboarding -> AI Business Audit -> newsletter -> subscription -> Today in Nexus return loop -> Hermes recommendations.

## Execution Priorities
1. Standardize CTA mapping from each content lane to one lead magnet.
2. Add onboarding branch by user intent (funding, automation, trading learning).
3. Add AI Business Audit handoff after onboarding completion.
4. Feed newsletter + Hermes daily recommendations from approved opportunities.
5. Track conversion events across each loop transition.

## Monetization Cohesion Rules
- Every opportunity requires CTA, lead magnet tie, and follow-up channel.
- Every newsletter item should map to one monetizable next action.
- Hermes recommendations should include conversion-aware next steps.

## Safety
- No aggressive unsolicited messaging.
- Keep explicit opt-in and bounded send policies.
