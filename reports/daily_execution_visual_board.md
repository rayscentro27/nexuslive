## Mobile Daily Board (Telegram Workflow)

```
TELEGRAM DAILY WORKFLOW CONCEPT
══════════════════════════════════════════════════════════════════════════════════

  MORNING SEQUENCE (7:00am):
  ─────────────────────────────────────────────────────────
  1. Send:  "good morning"
     Get:   Market brief + session context
  
  2. Send:  "what knowledge arrived?"
     Get:   Any new intel from email intake
  
  3. Send:  [your forex scan result — text summary]
     Get:   Hermes acknowledgment + any additional context

  ─────────────────────────────────────────────────────────
  MIDDAY CHECK (12pm):
  Send:  "open positions" or just check chart on paper platform
  (Hermes doesn't need midday check unless you have a question)

  ─────────────────────────────────────────────────────────
  EVENING CLOSE (4pm):
  Send:  "day recap: [brief summary]"
  Example: "day recap: 1 forex trade, +18 pips, EUR/USD long.
            AI offer card drafted. 1 contact listed. Score: 9/13."
  Get:   Acknowledgment + any follow-up

  ─────────────────────────────────────────────────────────
  WEEKLY (Sunday):
  CEO report auto-generated → review via Hermes
  "week recap" → triggers weekly summary
```
