## Crypto Market Cycle Framework

```
BITCOIN HALVING CYCLE (4-Year Rhythm)
──────────────────────────────────────────────────────────────────────

  PHASE 1: POST-HALVING (Month 1–6 after halving)
  → Supply shock begins, slow accumulation
  → Strategy: DCA into BTC/ETH, avoid altcoins
  → Signal: BTC dominance rising or stable

  PHASE 2: BULL RUN BUILD (Month 6–18)
  → Momentum builds, media attention increasing
  → Strategy: Hold core, add on dips, watch AI narrative leaders
  → Signal: Weekly closes above prior cycle ATH

  PHASE 3: ALTCOIN SEASON (Month 14–24)
  → Capital rotates from BTC → ETH → alts → AI tokens
  → Strategy: Hold BTC/ETH, add small AI narrative plays
  → Signal: ETH/BTC ratio rising, altcoin season index >75

  PHASE 4: DISTRIBUTION (Month 20–30)
  → Euphoria, mainstream media, parabolic moves
  → Strategy: Reduce alts, take profits on AI tokens, hold BTC
  → Signal: Retail FOMO, "this time different" narratives

  PHASE 5: BEAR / ACCUMULATION
  → 70–85% drawdown from ATH typical
  → Strategy: Long-term accumulation of BTC/ETH only
  → Signal: Negative sentiment, developer activity persisting

Current Phase: ______________ (update weekly)
```

## Crypto Setup Playbooks

### Setup A — BTC/ETH Cycle Position

```
WHEN TO ENTER:
  ✓ Post-halving accumulation window
  ✓ Price ≥ 20% below all-time high OR confirmed breakout above ATH
  ✓ Weekly RSI < 55 (not overheated)
  ✓ No extreme Fear & Greed (avoid >85 greed)
  ✓ BTC dominance stable or rising (for BTC entry)

METHOD:
  Dollar Cost Average (DCA) — not lump-sum
  Weekly or bi-weekly entries
  Never deploy full sim allocation at once

TARGETS:
  BTC: No hard target — hold until distribution signals appear
  ETH: Reduce position if ETH/BTC ratio exceeds prior cycle high
  
STOP / EXIT:
  Paper trading: Log paper entry and track against benchmark
  Exit signals: Weekly close below 20-week MA (Bitcoin Bear Warning)
  Distribution signal: >80 greed for 7+ consecutive days
```

### Setup B — AI Narrative Momentum Trade

```
ENTRY CRITERIA:
  ✓ Narrative score ≥ 32/50 (Tier 1) or ≥ 24/50 (Tier 2)
  ✓ Price is in early-stage breakout (not parabolic)
  ✓ Volume increasing over 7-day average
  ✓ Strong BTC market — no bear market entry for speculative tokens
  ✓ Position size: Max 5% of paper portfolio per AI token

ENTRY METHOD:
  Stage in: 50% initial, 50% on first confirmed pullback

TARGET:
  Tier 1: Hold for 2–4 months, then reassess narrative
  Tier 2: 30–50% profit target, take half off
  Tier 3: 2–3× or cut loss at 25% — no anchoring

STOP:
  Tier 2/3: Hard stop at 25% loss from entry
  Tier 1: Stop only if narrative fundamentally broken
```

### Setup C — Altcoin Season Rotation

```
SIGNAL:
  ✓ ETH/BTC ratio in confirmed uptrend (weekly)
  ✓ Altcoin Season Index > 75
  ✓ BTC dominance declining
  ✓ 30-day rolling market up ≥ 20%

EXECUTION:
  Rotate from BTC → ETH first (less speculative)
  Then rotate small % into AI narrative Tier 1 tokens
  Keep 60%+ in BTC/ETH at all times

NO-TRADE RULE:
  Never rotate into alts if BTC dominance is RISING
  Never chase alts after they are already up >100% in 30 days
```
