# Business MVP Framework

Date: 2026-05-11

## Objective
Provide ethical, rapid Netlify-ready MVP scaffolds for opportunity validation.

## MVP Template Families
1. Lead Capture Landing Page
- Hero, value proposition, proof block, CTA form, compliance footer.

2. Educational Microsite
- Topic hub, explainer modules, gated checklist, newsletter CTA.

3. Opportunity Directory
- Filterable listings, trust badges, source links, owner disclosure.

4. Service Offer Page
- Problem/solution narrative, process timeline, intake CTA.

## Generator Blueprint
- Input: niche, audience, offer type, CTA goal, risk/compliance profile.
- Output:
  - page copy skeleton
  - component map
  - visual direction tokens
  - deployment checklist

## Netlify Deployment Path
- Static output conventions (`public/`, metadata, redirects, forms).
- Manual deployment/approval only.
- Analytics hook points for funnel visibility.

## Ethical Constraints
- No brand cloning.
- No copyrighted copy reuse.
- No deceptive testimonials or fabricated proof.
